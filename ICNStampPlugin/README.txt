ICNStampPlugin - Ready-to-Import ICN Plugin Practice Project

Features:
- Adds "Stamp Document" action to the document context menu in ICN.
- Calls a custom service to stamp documents with 'StampedBy' = current user.
- Displays confirmation on completion.

Usage:
1. Import into Eclipse.
2. Add navigatorAPI.jar to the build path (do not export).
3. Export as JAR.
4. Upload via ICN Admin (Plugins).
5. Add action to Document Context Menu in ICN.
6. Test by right-clicking a document and selecting "Stamp Document".
